# 🚀 Advanced JavaScript Landing Page

A modern responsive landing page built as part of **The Complete JavaScript Course** by Jonas Schmedtmann.

This project focuses on mastering **advanced DOM manipulation techniques** and building highly interactive user interfaces using **vanilla JavaScript**.

---

## 🌐 Live Demo

👉 **[View Live Project](https://mashraf114.github.io/Landing-Page)**

---

## 📌 Project Overview

This landing page simulates a real production-ready marketing website while emphasizing **JavaScript-driven UI behavior** instead of static design.

The main goal was to deeply understand how JavaScript controls the browser, user interaction, and dynamic interface updates.

Unlike traditional landing pages, most interactions here are powered entirely by JavaScript logic.

---

## Core Learning Focus

This project was built specifically to practice **Advanced DOM Manipulation**, including:

- Efficient DOM selection & traversal
- Event delegation
- Smooth scrolling navigation
- Sticky navigation using Intersection Observer API
- Lazy loading images
- Reveal sections on scroll
- Tabbed component implementation
- Slider / Carousel component
- Dynamic class manipulation
- Performance optimization techniques
